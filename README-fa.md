# 🖥️ اسکریپت مانیتورینگ سلامت سرور لینوکس

یک اسکریپت Bash سبک و کارآمد که برای مدیران سیستم (SysAdmins) و مهندسان DevOps طراحی شده است تا وضعیت منابع حیاتی سرورهای لینوکسی را زیر نظر بگیرد. این اسکریپت میزان مصرف پردازنده (CPU)، رم، فضای هارد دیسک و وضعیت سرویس‌های مهم را بررسی کرده و در صورت عبور از آستانه مجاز (Threshold)، به صورت خودکار یک هشدار به ربات تلگرام شما ارسال می‌کند.

### 🚀 امکانات
- **مانیتورینگ پردازنده و رم:** ارسال هشدار در صورت عبور مصرف منابع از حد تعیین شده (مثلاً بالای ۸۵ درصد).
- **بررسی فضای دیسک (Disk Space):** جلوگیری از قطعی سرور به دلیل پر شدن هارد با ارسال هشدار پیش از رسیدن به ۱۰۰ درصد.
- **مانیتورینگ سرویس‌ها:** بررسی مداوم سرویس‌های حیاتی مانند `Nginx`، `Apache` یا `MySQL` برای اطمینان از روشن بودن آن‌ها (Running).
- **اتصال به تلگرام:** ارسال آنی هشدارها (Push Notifications) به موبایل یا سیستم شما از طریق API تلگرام.

### 🛠️ پیش‌نیازها
- یک سیستم‌عامل مبتنی بر لینوکس (اوبونتو، سنت‌اواس، دبیان و غیره)
- نصب بودن ابزار `curl` (کد نصب: `sudo apt install curl`)
- یک توکن ربات تلگرام (ساخته شده از طریق BotFather@)
